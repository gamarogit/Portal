# Sistema de Entrenamiento

Microservicio de gestión de capacitaciones y entrenamientos.

## Puertos
- Backend: 3002
- Frontend: 3102

## Inicio Rápido

```bash
# Backend
cd backend && npm install && npm run start:dev

# Frontend  
cd frontend && npm install && npm run dev
```

## Integración
Este microservicio se integra con el portal principal en `activos-portal`.
